# TUI használat

A TUI a CLI fölé épített interaktív réteg.

Indítás:

```bash
  npm run tui
```

## Jelenlegi nézetek

- pipeline áttekintő,
- teljes pipeline futtatása,
- ICS generálás,
- összes audit futtatása.

## Billentyűk

- `↑` / `↓` — választás
- `Enter` — indítás
- `Esc` vagy `v` — vissza a menübe
- `q` — kilépés
